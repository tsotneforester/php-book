Dripal Modules

  • Create NewModule folder in drupal > modules > custom
  • Create NewModule.info.yml file
  • Now it can be found in Drupal Extends, Install it
  • Create src folder in NewModule folder to create controller php file
  • Create NewModule.routing.yml in NewModule folder
  • now on drupalproject/herolist you can see content from controller
  • use Drupal\Core\Controller\ControllerBase; - if you add this class to your controller, it will have some usefull methods
  • Create NewModule.module file to hook theme instead of hook you should use module name
  • Then you create temlate folder with twig file in it
  • Simplify controller.php to do end with same result
  • Twig Language 2 - VS Code Extension for .twig file sintax vizualization
  • Drupal Syntax Highlighting - VS Code Extension for .module file sintax vizualization
