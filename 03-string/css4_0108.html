substr()

The substr() function returns a part of a string.
$rest = substr ("abcdef" , -1); // returns "f"
$rest = substr ("abcdef" , -2); // returns "ef"
$rest = substr ("abcdef" , -3 , 1); // returns "d"
$rest = substr ("abcdef" , 0 , -1); // returns "abcde"
$rest = substr ("abcdef" , 2 , -1); // returns "cde"
$rest = substr ("abcdef" , 4 , -4); // returns ""
$rest = substr ("abcdef" , -3 , -1); // returns "de"

String Position

strpos() - Finds the position of the first occurrence of a string inside another string (case-sensitive)
stripos() - Finds the position of the first occurrence of a string inside another string (case-insensitive)
strrpos() - Finds the position of the last occurrence of a string inside another string (case-sensitive)
strripos() - Finds the position of the last occurrence of a string inside another string (case-insensitive)
(string,find,start)
$title = "stairway to haven";
echo stripos ($title , "a"); // 2
echo stripos ($title , "A"); // 2
echo strpos ($title , "A"); //
echo strpos ($title , "a"); // 2
echo strrpos ($title , "A"); //
echo strripos ($title , "a"); // 13

String Comparison


PHP provides several functions that let you test whether two strings are approximately equal
soundex()
metaphone()
similar_text()
levenshtein()

$a = "Fred";
$b = "Phred";
if (soundex ($a)== soundex ($b)) {
echo "Sound Equally";
}
else{
echo "Not Equally";
}//Not Equally
if (metaphone ($a)==metaphone($b)) {
echo "Sound Equally";
}
else{
echo "Not Equally";
}//Sound Equally
//The Metaphone algorithm is generally more accurate than soundex
$string1 = "Rasmus Lerdorf";
$string2 = "Razmus Lehrdorf";
$common = similar_text ($string2 , $string1 , $percent);
printf ("They have %d chars in common (%.2f%%)." , $common , $percent);
//They have 13 chars in common (89.66%).
printf ("They have %d chars in common (%d%%)." , $common , $percent);
//They have 13 chars in common (89%).
//Be aware when using this function, that the order of passing the strings is very important if you want to calculate the percentage of similarity, in fact, altering the variables will give a very different result
echo $similarity = levenshtein ($string1 , $string2); // 2
//The Levenshtein distance is defined as the minimal number of characters you have to replace, insert or delete to transform string1 into string2.

substr_count()

To learn how many times a smaller string occurs within a larger one,
use substr_count():


$sketch = <<< EndOfSketch
Well, there's egg and bacon; egg sausage and bacon; egg and spam;
egg bacon and spam; egg bacon sausage and spam; spam bacon sausage
and spam; spam egg spam spam bacon and spam; spam sausage spam spam
bacon spam tomato and spam;
EndOfSketch;

$count = substr_count ($sketch , "spam");
print ("The word spam occurs {$count} times.");
//The word spam occurs 14 times.

string padding

The str_pad() function pads a string to a new length.
$input = "Alien";
echo str_pad ($input , 10); //"Alien " strlen() = 10
echo str_pad ($input , 10 , "-=" , STR_PAD_LEFT); //"-=-=-Alien"
echo str_pad ($input , 10 , "_" , STR_PAD_BOTH); //"__Alien___"
echo str_pad ($input , 6 , "___"); //"Alien_"
echo str_pad ($input , 3 , "*"); //"Alien"
