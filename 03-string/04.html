$str = "A 'quote' is <b>bold</b>";
print (htmlentities ($str));
print (htmlspecialchars ($str));
The HTML output of the codes above will be the same (View Source):
"A 'quote' is &lt;b&gt;bold&lt;/b&gt;"
The browser output of the code above will aslo be the same:
"A 'quote' is <b>bold</b>"
Unlike htmlspecialchars , htmlentities converts not only special characters in the string to HTML entities but all applicable characters to HTML entities.
The strip_tags()function removes HTML tags from a string:
$input = '<p><b>Howdy</b>, &quot;Cowboy&quot;</p>';
echo $output = strip_tags ($input);
'Howdy, &quot;Cowboy&quot;'
echo $output = strip_tags ($input , '<b>');
'<b>Howdy</b>, &quot;Cowboy&quot;'
