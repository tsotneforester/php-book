Setup SCSS

  • Instal node.js
  • instal npm from cmd - "npm install -g npm"
  • install sass from cmd - "npm install -g sass"
  • Install Vscode extention "Live Sass Compiler"
  • Create "style.scss"
  • Click the ‘watch sass’ button on the bottom right of your VSCode screen.
  • The compiler will automatically create two additional files in your directory. styles.css and styles.css.map
  • Now we are ready to start writing SCSS code in the styles.scss document which will be converted to css by live sass compiler each time we hit save.
