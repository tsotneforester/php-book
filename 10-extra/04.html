Easy Mode / Advanced Mode

  • node.js and nmp must be installed
  • npm i -D tailwindcss - installs tailwindcss in working directory
  • npx tailwindcss init - creates tailwind.config.js, where you must edit content, to specify files with tailwind classes

  • create input.css with @tailwind directives
  • npx tailwindcss -i ./input.css -o ./output.css --watch - specify input.css location as input (-i) and output.css location as output (-o)
  • npm init -y - creates package.json, were in "scripts" you put above npx text and every time instead of hard script, you npm run build
